# Contributing

Thank you for your interest in contributing.

## How to Contribute

1. Open an issue to discuss significant changes.
2. Create a branch for your work.
3. Make focused changes.
4. Add or update tests and documentation when appropriate.
5. Open a pull request.

## Pull Requests

Pull requests should include:

- A clear description of the change
- The reason for the change
- Any relevant issue links
- Notes about testing

## Issues

When opening an issue, include:

- What happened
- What you expected to happen
- Steps to reproduce, if reporting a bug
- Any relevant logs, screenshots, or context

## Code Style

Follow the style and conventions already used in the repository.

## Review

Maintainers may request changes before merging.

Please keep discussions respectful and focused on improving the project.
